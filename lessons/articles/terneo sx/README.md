:ballot_box_with_check: Пакадж показанный в видео:    

```yaml
terneo:

    homeassistant:

      customize:
        automation.set_terneo_temp:
          friendly_name: Установка температуры Terneo
          icon: mdi:thermostat

    climate:
      - platform: terneo
        serial: 'SERIAL NUMBER'
        host: 'IP ADRESS'

    automation:
    
      - alias: set_terneo_temp
        initial_state: true
        trigger:
          - platform: state
            entity_id: input_button.terneo
        action:  
          - service: mqtt.publish
            data_template:
              topic: "terneo/terneo sx_D61D70/set/setTemp"
              payload_template: "{{states('input_number.terneo')}}"
              retain: true 
```
